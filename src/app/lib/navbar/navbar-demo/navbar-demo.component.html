Navbar

The navbar is a wrapper that positions branding, navigation, and other elements in a concise header.

Home Buttons
  • Health Catalyst
  • Health Catalyst Community 
{{navbar}}
uri's are purposely left undefined for demo purposes

Properties

  • user - current user
  • appIcon - path to svg
  • homeUri - RouterLink uri to home page
  • fixedTop - statically positions the navbar to the top of the page see below
  • hc-navbar-link - main navigation links
    • active (optional) - forces active state (default = null)
    • uri - RouterLink uri
    • exact - sets the RouterLinkActive options to match the url exactly to set active state. (default = false)
  • hc-navbar-icon - icons and buttons displayed on the right side
  • hc-app-switcher - include app switcher functionality, if this element is not present, it will revert to navigating to the home page

App Switcher

Include the following markup within hc-navbar 
<hc-app-switcher></hc-app-switcher>
setup is required for the module 
{{moduleSetup}}
