Vertcal Tab Usage
Vertical Tabs are most often used as a form of secondary navigation on pages that have a large number of sub-pages. For those with a fixed, smaller number of sub-pages, using Horizontal Tabs might be more appropriate.

Horizontal Tab Usage
Horizontal tabs can be used anywhere, but are most commonly used as a secondary navigational elements on pages that have a limited number of sub pages. Those with long lists of sub pages are better served using Vertical Tabs.

Together
Horizontal and Vertical tabs may be used in conjunction on pages where secondary and tertiary levels of navigation are required.



Angular Component

        
            <hc-tab-set direction="horizontal">
            	<hc-tab title="Tab 1">
            		Tab content 1
            	</hc-tab>
            	<hc-tab title="Tab 2">
            		Tab content 2
            	</hc-tab>
            </hc-tab-set>


Component Parameters

title Type: String On Tab Component. Use to change the title of the tab
direction Type: 'vertical' | 'horizontal' (default: 'vertical') On Tab Set component. Determines whether horizontal or vertical tabs.
