Dropdown Menu

Menus display a list of choices with one choice per line.

Standard Dropdown Menu

Dropdown Types
There are two primary types of dropdowns used in Cashmere: one is an extension of the button component, and the other is this dropdown component. The guidelines for which dropdown to use is based off what items are contained in the dropdown. The button dropdown is for a list with action items (e.g. delete all selected items). If the dropdown contains a list of selectable items, then this component should be used.
