Checkbox

Checkboxes allow the user to make a binary choice, i.e. a choice between one of two possible mutually exclusive options.

Standard
Standard Checkbox
Disabled
Disabled Checkbox

Checkbox Usage
Checkboxes are typically used with labels, but can be used without in certain situations (table columns, etc).
This checkbox can be used with both reactive and template forms



Angular Component

          
            <hc-checkbox id="checkbox-id">Checkbox Label</hc-checkbox>
            <hc-checkbox id="checkbox-id" [(ngModel)]="data">Checkbox Label</hc-checkbox>


Component Parameters

id Type: String Optional - checkbox identifier
disabled Type: string|boolean Optional - disables the checkbox
required Type: string|boolean Optional - whether the checkbox is required
name Type: string Optional - name to be applied to input element
value Type: string Optional - value attribute for the input element
tabIndex Type: number Optional - tabIndex for the element
checked Type: number Optional - whether the checkbox is checked or not
